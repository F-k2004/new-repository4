## 🚀 درباره پروژه

- کار: نمایش اطلاعات پروژه مثل نویسنده، تاریخ و توضیحات

1. ابتدا مخزن (repository) رو کلون کن یا دانلود کن.
2. دستور زیر رو اجرا کن:

```bash
python project_info.py
```
